# elasticity notes
Notes by GB on relevant concepts in membrane elastic theory

These notes attempt to follow an algebraic path as much as possible, with the minimal possible amount of differential geometry. They provide more background and intermediate steps than a text like Safran.  

Each file covers a different topic; some are noted as "ingredients" because they contain important background information rather than concepts specific to membranes.
